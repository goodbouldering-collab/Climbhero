# SEO最適化 必須要件メモ

## 🎯 常に意識すべきSEO最適化項目

このプロジェクトでは、以下のSEO最適化を**必ず**実装してください：

### 1. **JSON-LD構造化データ**
- Schema.org準拠の構造化データを実装
- VideoObject、Article、WebPage、Organization、Breadcrumbなど適切なタイプを使用
- 検索エンジンがコンテンツを理解しやすくする

### 2. **OGP (Open Graph Protocol)**
- Facebook、Twitter、LINEなどSNSでの表示最適化
- og:title, og:description, og:image, og:urlは必須
- Twitter Card対応 (twitter:card, twitter:title等)

### 3. **LLMO (Large Language Model Optimization)**
- AI検索エンジン（ChatGPT、Perplexity等）対応
- ナレッジグラフ設計による情報構造化
- セマンティックHTML活用
- メタデータの充実化

### 4. **基本的なSEO要件**
- 適切なtitleタグ（60文字以内）
- meta descriptionの最適化（120-160文字）
- canonical URL設定
- robots.txt、sitemap.xml
- レスポンシブデザイン対応
- ページ読み込み速度最適化
